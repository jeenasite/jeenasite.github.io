Design Outline

Type System

- Type Levels
- Font Selection

The type system to be applied to the website will focus on highlighting the differences in text importance in terms of its layering on the website. To achieve this, the type system will incorporate 4 text sections which will include:

- Heading: This will be the representation of all text deemed as the heading. As an example, in this document, Type System, Color System, Navigation and Layout are headings

- Subheading: This will be the representation of all text deemed as the subheading: As an example, in this document, Type level, Font Selection, Mood and Color Palette are subheadings.

- Menu: This will be representation of all text deemed as a menu. As an example, for the website will contain a menu with links to the home page and the two supplementary pages

- Body: This will be a representation of all text deemed as the body. As an example, for the website, the chosen text will be part of the body. These four sections will consist of levels described below:

Type levels

Heading 1 – This will denote the main heading

Subheading - Heading 2 – This will be the subheading denoting any text where the heading is Heading 1.

Subheading - Heading 3 – This will be the subheading denoting any text where the heading is Heading 2

Subheading - Heading 4 – This will be the subheading denoting any text where the heading is Heading 3

Menu - The menu will only contain 1 level

Body- The body will only contain 1 level

Font Selection

Two fonts will be selected which will include

- Primary Font – this will be the font to be loaded by default. The font files for this font will need to account for all the levels of each section.

- Fallback Font – this will be the font to be used incase the default font does not load as expected. The font files for this font will also need to account for all the levels of each section.

Color System

- Mood
- Color Palette

The design of the website will apply the use of color to communicate mood and emotion after which a color palette will be developed to determine how colors will be mixed. The end goal is to have an analogous mix of colors where related colors are used with one dominant color and several supporting colors.

Mood

The following table shows the mood mapped out to the color that is mostly associated with it.

Mood Color
Horror / Love Red
Thrill Orange
Happiness Yellow
Growth Green
Trust Blue
Luxury Purple
Cleanliness White
Mystery Black

Color palette

To generate a color palette, the identified colors in the mood section will be mixed in terms of their place on a rank that segments colors as primary, secondary, tertiary or other. The ranking is shown below:

- Primary Colors: Blue, Yellow and Red

- Secondary Colors: Green, Purple and Orange

- Tertiary Colors: White

- Other: Black

Navigation

Navigation will be primarily be on the menu section with each menu component being linked to each other through backlinks. The type of navigation will be reverse navigation where a user on any of the three pages can go to any of the pages regardless of their current location.

Layout

A responsive layout will be applied to allow the website pages react and adjust to different screen types. The screen types to be considered will be:

- Desktop / Laptop screen

- Tablet Screen

- Mobile Screen
